## Key Layers
- **Hardware**: 3x Dell Edge + Netgear Switch + TP-Link Extender
- **OS**: Ubuntu 22.04.5 LTS
- **Orchestration**: k3s Lightweight Kubernetes
- **GitOps**: ArgoCD (managing AdGuard + planned for all apps)
- **Storage**: Local Path Provisioner
- **Applications**:
  - Home Assistant + Philips Hue
  - AdGuard Home (network-wide ad blocking + DNS)
  - Prometheus + Grafana + Node Exporter
- **Access Pattern**: NodePort services
- **Future**:
  - Full App of Apps pattern in ArgoCD
  - Ingress + Cert-Manager
