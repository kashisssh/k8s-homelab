## Key Layers
- **Hardware**: 3x Dell Edge Platform Nodes + networking
- **OS**: Ubuntu 22.04.5 LTS
- **Orchestration**: k3s (lightweight Kubernetes)
- **Storage**: Local Path Provisioner
- **Applications**:
  - Home Assistant + Philips Hue
  - AdGuard Home (Ad Blocking + DNS)
  - Monitoring Stack (Prometheus + Grafana + Node Exporter)
- **Access**:
  - Home Assistant: NodePort 30123
  - AdGuard Home: NodePort 30080
  - Prometheus: NodePort 30990
  - Grafana: NodePort 31000
- **Future**:
  - Ingress Controller
  - GitOps with ArgoCD
  - Loki + Promtail (logging)
